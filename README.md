# Mon Projet Unity

Ce projet est un jeu développé avec Unity, créé pour tester différentes mécaniques de jeu en 3D, comme le tracking d'objets et les jeux de tir à la cible.

## Description

Le jeu propose plusieurs "stands" où le joueur peut tester ses compétences :

- **Tracking Stand** : Le joueur doit suivre un objet qui se déplace dans une zone spécifiée.
- **Shooting Stand** : Le joueur clique sur des cibles qui apparaissent dans une zone, et son score augmente chaque fois qu'il touche une cible.

Chaque stand a une durée de 30 secondes, et le score est mis à jour en temps réel pendant le jeu.

## Fonctionnalités

- Création dynamique de cibles dans des zones spécifiques.
- Système de score qui se met à jour en fonction de l'action du joueur.
- Interface avec des boutons interactifs dans le monde du jeu.
- Suivi du joueur avec une caméra FPS.

## Installation

1. Clonez ce repository ou téléchargez-le en ZIP.
2. Ouvrez le projet Unity avec votre version d'Unity (recommandée : Unity 2020.x ou version supérieure).
3. Exécutez le projet dans l'éditeur Unity.
4. la sceene est aimlab.unity

## Prérequis

- Unity 2020.x ou version supérieure.
- Aucun package externe requis pour ce projet.

## Utilisation

1. Cliquez sur le bouton "Start Game" pour démarrer le jeu.
2. Suivez l'objet ou cliquez sur les cibles pour accumuler des points.
3. Le jeu se termine après 30 secondes, et votre score final est affiché dans la console Unity.
